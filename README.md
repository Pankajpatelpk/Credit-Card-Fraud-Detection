# 💳 Credit Card Fraud Detection System

A Machine Learning-based Credit Card Fraud Detection web application that predicts whether a credit card transaction is **Fraudulent** or **Legitimate** using an **XGBoost Classifier**. The application is built with **FastAPI** for the backend and **HTML, CSS, and JavaScript** for the frontend.

---

## 📌 Project Overview

Credit card fraud is one of the biggest challenges in digital banking. This project uses a trained XGBoost machine learning model to analyze transaction details and classify transactions as **Fraud** or **Not Fraud** in real time.

The application provides a simple and user-friendly interface where users can enter transaction information and instantly receive the prediction.

---

## 🚀 Features

- 🔍 Real-time fraud prediction
- ⚡ FastAPI backend
- 🤖 Machine Learning model trained with XGBoost
- 🔄 JSON-based API communication
- 🌐 CORS enabled for frontend-backend integration
- 📊 Instant prediction results
